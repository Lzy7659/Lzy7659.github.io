目标

准备搭建环境

1. node.js安装

  1. 下载地址：node.js
  2. 直接安装，不需要改动
  3. 检查安装成功与否：

6. 体验hexo

  1. 输入：

  2. 将生成的：http://localhost:4000/ 输入浏览器就可以本地查看

    7.将hexo与github pages联系起来(重点)

  3. 配置git相关信息

  4. 接下来如不需密码，三次回车即可

  5. 将ssh key 添加到github中
    打开C:\Users\用户名.ssh，将id_rsa.pub中的公钥全部复制，到github的settings中，创建new key，保存即可

  6. 检查成功与否，输入ssh -T git@github.com若提示欢迎即为成功
    在此步出现问题，ssh始终无法连接成功
    将MyBlog文件夹中_config中 

    deploy:
      type: git
      repo: git@github.com:yourname/yourname.github.io.git
      branch: master

    改为

    deploy:  
    type:git  
    repo:https://github.com/Lzy7659/Lzy7659.github.io.git  
    branch: master

    8.发布文章

    MyBlog中打开命令行，输入：hexo new "article title"
    生成部署之前需要安装扩展，输入：npm install hexo-deployer-git --save
    继续命令行输入

    hexo g //生成
hexo d //部署
